# Common Tasks

## Database
```bash
npx prisma migrate dev    # Create migration
npx prisma generate       # Generate client
npx prisma studio         # Open GUI
```

## Development
```bash
pnpm dev                  # Start dev
pnpm build                # Build
pnpm lint                 # Lint code
```

## Components
```bash
npx shadcn-ui add button  # Add component
```
